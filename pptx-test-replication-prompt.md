# PowerPoint Strategy Deck Replication Prompt

## Overview
Create a comprehensive 19-slide strategic planning presentation deck with a professional blue color scheme. This deck is designed for presenting corporate or organizational strategy in a structured, visually cohesive format.

## Design System

### Color Palette
**Primary Colors:**
- Deep Blue (Accent1): `#051C2C` - Main brand color, used for headers and key elements
- Bright Blue (Accent2): `#22A3DF` - Supporting accent color
- Electric Blue (Accent3): `#2222F6` - Additional accent
- Teal (Accent4): `#1877A2` - Used for strategic initiative boxes
- Light Blue (Accent5): `#A2D5EC` - Light accents
- Gold (Accent6): `#F8BC3B` - Highlight color for version badges

**Supporting Colors:**
- Black: `#000000`
- White: `#FFFFFF`
- Dark Gray: `#4D4D4D`
- Mid Gray: `#7F7F7F`
- Light Gray: `#B3B3B3`, `#D0D0D0`, `#E6E6E6`

**Gradient Colors (for linear scales):**
- Linear 1 (Deep Blue): `#051C2C`
- Linear 2: `#034B6F`
- Linear 3: `#027AB1`
- Linear 4 (Cyan): `#00A9F4`
- Linear 5: `#39BDF3`
- Linear 6: `#71D2F1`
- Linear 7 (Pale Blue): `#AAE6F0`

### Typography
- **Primary Font:** Arial (all text)
- **Title Size:** 14pt (typically bold)
- **Body Text Size:** 14pt
- **Small Text Size:** 16pt for version badges
- **Alignment:** Centered for most strategic elements, left-aligned for bullet lists

### Slide Dimensions
- **Standard 16:9 format** (13.33" × 7.5" or 720pt × 405pt)

## Common Design Patterns

### Version Badges
- Appear on slides 1-4 in top right corner
- Gold/cream background (`#F8BC3B` with 80% tint)
- Black dashed border
- 16pt text, centered
- Dimensions: approximately 1.79" × 0.44"

### Title Placement
- Consistent across all slides
- Top of slide, typically left-aligned or centered
- Deep Blue text on white background or white text on Deep Blue background

### Shape Styling
- **No borders** on most shapes (noFill on borders)
- **Rounded corners** on some rectangular elements
- **Center-aligned text** for strategic elements
- **Consistent padding** in text boxes (lIns="91425" tIns="45700" rIns="91425" bIns="45700" in XML)

### Text Formatting
- **Bold** for headers and important items
- **Regular weight** for body text and descriptions
- **White text** on dark backgrounds (Deep Blue, Teal)
- **Dark text** on light backgrounds

### Box Hierarchy
- **Large boxes** for high-level concepts (vision, objectives)
- **Medium boxes** for initiatives and key elements
- **Small boxes** for details and supporting information

## Content Patterns

### Placeholder Text Style
- Square brackets for replaceable content: `[insert description]`
- "xx" for numerical placeholders
- "202x" for year references
- Descriptive context within brackets to guide replacement

### Common Placeholder Examples
- `[insert title/name of strategy, if applicable]`
- `[Insert purpose and/or vision]`
- `[Insert strategic objective #1]`
- `[Insert strategic initiative]`
- `[Insert key enabler(s)]`
- `[insert description]`

- `[currency]` for monetary values

## Implementation Instructions

When recreating this deck:

1. **Start with theme setup**: Apply the color scheme and Arial font throughout
2. **Create master elements first**: Set up the title format, version badge style, and common shape styles
3. **Build slides in order**: Each slide builds on previous concepts
4. **Maintain exact spacing**: The visual hierarchy depends on consistent sizing and positioning
5. **Use white text on colored backgrounds**: Ensures readability and professional appearance
6. **Keep borders minimal**: Most shapes have no visible borders for clean, modern look
7. **Center-align strategic content**: Makes the deck feel balanced and authoritative
8. **Test with real content**: Replace placeholders with actual strategy content to verify layout scales properly

## Key Design Principles

1. **Visual Hierarchy**: From broad vision to specific initiatives to foundational elements
2. **Color Consistency**: Blue palette throughout creates cohesive brand
3. **White Space**: Not overcrowded despite complex information
4. **Centered Elements**: Creates formal, balanced appearance suitable for executive presentations
5. **Modular Boxes**: Consistent box sizing within sections allows for easy content swapping
6. **Professional Typography**: Simple Arial font maintains readability at presentation scale
